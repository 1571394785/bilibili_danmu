## 使用方法
### 1.安装crx
### 2.打开任意视频网站，如<a href="http://www.yinghua8.com/">樱花动漫</a>，风车动漫
### 3.点击插件图标，选择导入弹幕
### 4.选择加载弹幕，并点击测试
### 5.选择元素，定位弹幕位置
**注意：因为目前还在开发，功能仍然不完善**
